# WebsitePariwisataNabilAlifah
Website Exploration adalah platform informasi destinasi wisata di Yogyakarta yang dirancang untuk memudahkan pengguna dalam menemukan tempat wisata populer seperti pantai, taman rekreasi, tempat edukasi, hingga area perbelanjaan terkenal. Website ini memiliki dua role utama yaitu User dan Admin, masing-masing dengan fitur berbeda.
